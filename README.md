# Radio de Voladura UNACEM - Versión 18

Cambios:
- Mantiene branding UNACEM de la versión 15.
- Permite 2 voladuras por día.
- Cada voladura tiene coordenadas UTM X/Y independientes.
- Cada voladura tiene ángulo de rotación horario independiente.
- Corrige la geometría: ya no usa círculo simple ni polígono tipo escudo.
- Dibuja una plantilla compuesta por dos arcos y brazos laterales:
  - Rojo: sector de riesgo para equipos, R=300 m.
  - Verde: radio de evacuación de personal, R=500 m.
- La plantilla completa se rota sin modificar dimensiones.
