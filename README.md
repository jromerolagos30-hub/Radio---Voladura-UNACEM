# Radio de Voladura UNACEM - Versión 18

Versión final con geometría de radios corregida según bosquejo aprobado.

## Cambios principales
- Se mantiene el centro de cada voladura como origen geométrico.
- Radio rojo de personas/equipos de riesgo: 150 m.
- Radio verde de evacuación/personal: 300 m.
- Contorno rojo corregido con forma semicircular inferior y sector superior integrado.
- Contorno verde corregido como envolvente de 300 m.
- Dos voladuras por día con ángulo de giro independiente: 15° y 195°.
- Vista tipo dashboard lista para subir a GitHub Pages.

## Uso en GitHub Pages
1. Subir `index.html` y `styles.css` a un repositorio.
2. Activar GitHub Pages desde Settings > Pages.
3. Seleccionar la rama `main` y carpeta `/root`.
