# Radio de Voladura UNACEM - Versión 17

- Basado en versión 15 con branding UNACEM.
- Permite hasta 2 voladuras por día.
- Cada voladura tiene coordenadas UTM y ángulo de giro horario.
- Usa polígonos tipo plantilla: rojo = sector de riesgo para equipos, verde = radio de evacuación de personal.
- Usuario final valida GPS contra polígonos activos.
