# Radio de Voladura UNACEM - Versión 16

Actualización solicitada sobre la versión 15:

- Se mantiene el branding UNACEM y se conserva el retiro del botón PORTADA.
- El administrador ahora puede registrar hasta 2 voladuras por día.
- Cada voladura tiene coordenadas UTM independientes:
  - Coordenada de primera voladura.
  - Coordenada de segunda voladura.
- Cada voladura tiene su propio campo de ángulo de giro / rotación en grados.
- La geometría del radio no cambia sus dimensiones; solo rota según el ángulo ingresado.
- La vista de usuario final muestra las voladuras activas, sus horarios, ángulos y radios.
- La evaluación GPS considera ambas voladuras:
  - Rojo: sector de riesgo para equipos.
  - Verde: radio de evacuación de personal.
