Uso recomendado

  1. El responsable ingresa las coordenadas UTM del punto de voladura.
  2. Define radio de equipos y personal.
  3. Comparte el enlace o QR.
  4. El usuario activa GPS y recibe el estado: seguro, alerta o evacuar.

Nota: Este piloto es referencial. Para operación real debe validarse con Topografía/GIS y con el procedimiento de voladura de la empresa.
