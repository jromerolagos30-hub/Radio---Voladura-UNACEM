Administrador de Voladura

Usa un plano en la nube para que todos vean la misma versión desde cualquier celular.

Admin

1. Registrar / actualizar datos

Recomendado: subir el plano a SharePoint/OneDrive/Google Drive y pegar un enlace con permiso de visualización para cualquier persona con el vínculo.

Cómo obtener la URL del plano

SharePoint / OneDrive: subir plano → Compartir → “Cualquier persona con el vínculo puede ver” → copiar enlace.

Google Drive: subir plano → Compartir → acceso general “Cualquier persona con el enlace” → copiar enlace.

2. Enlace único para usuarios

El enlace incluye fecha y hora en la consulta: ?v=AAAAMMDD-HHMM.
